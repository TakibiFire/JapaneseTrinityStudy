# 日本版リタイア後の取り崩し戦略 〜 4%ルールを信じるな

「4%ルール」はご存知でしょうか。これは、「資産運用をしながら4%を取り崩して生活すれば、30年以上経過しても資産が底をつく確率は極めて低い」という経験則のことです。アメリカのトリニティ大学の研究者たちが提唱したため、「トリニティ・スタディ」とも呼ばれます。

==しかしこの4%ルールを信じて資産取り崩し戦略を立てると取り返しのつかないことになりかねません。==

## 第1章: 結論

### 資産形成と取り崩しは違う

資産を増やす段階（資産形成）と、資産を使っていく段階（取り崩し）では、取るべき行動が変わります。

資産形成においては、リターンを大きくし、インデックスファンドを長期保有することが重要です。運用中に価格が下落しても、働き続けて入金を続ける限り、時間の経過とともに回復を見込めます。

しかし、取り崩しの段階では状況が異なります。資産を取り崩しながら運用を続ける場合、運用初期に価格が下落すると、資産が目減りした状態から生活費を引き出すことになります。その結果、資産の枯渇が早まり、回復のスピードも遅くなります。この現象は[収益率配列のリスク](sequence.md)と呼ばれ、運用初期の不調がその後の資産寿命に致命的な影響を与えます。これを避けるためには、資産形成時とは異なるアセットアロケーション（資産配分）を組むことが必要になります。

### 最適な資産配分は年齢と目標年数で変わる

まず結論として、日本において「オルカンに100%投資し、毎年4%を取り崩す」というルールを実行した場合、物価上昇や税金などを考慮すると、30年後の破産確率は 35.4%、50年後の破産確率は 58.7% に達します。「4%なら安全」という前提は成り立ちません。

現実的には「現在$A$歳のあなたの総資産が$X$円で、現在の年支出が$Y$円の時、今から$N$年間資産が枯渇しない確率は$P$%である」ということがシミュレーションで求まります。つまり5つの数字の関係で成り立っていて、このうち、4つを決めると最後の一つの数字が決まるという性質があります。

仮にあなたが50歳として総資産が1億とします。後残りの数字が3つなので、「85歳まで枯渇しない確率を90%にしたい」と決めると年支出$Y$が求まります。逆に「今の生活の年400万円支出は維持しながら90%行きられる安全策で行きたい」と決めると何年行きられるか$N$が求まります。

4%ルールはこれらを完全に無視して $Y/X = 4\% $ なら大丈夫！という現実には沿わない理論となっています。

**何年生きることを目標とするか: $N$の重要性**

重要なことに、「何年生きたいか」で最適戦略が変わるケースがあります ([無リスク資産を持つ](zero_risk.md) などで触れます)。例えば複数資産を運用する場合、後20年生存したい人と40年生存したい人とで、どの順で切り崩すかという最適解が逆になる場合があります。

**あなたが何歳か: $A$の重要性**

あなたが若いかどうかで戦略も変わってきます。若ければ若い FIREをするほど目標年数 $N$ を大きくしなければならず、一般的には厳し目の生存確率$P$が出ます。

平均的な2人以上の世帯では支出が55歳がピークになっているので、人によってはこれから出費がかさむことを考慮する必要があるでしょう。
60歳で国民年金保険料の支払いが終わり、年金をいつから受給するかという選択があります。

こういう側面を考慮すると、現在の年齢はかなり最適戦略に関わってきます。

**生存確率 $P$の重要性**

モンテカルロ・シミュレーションをすると、「資産運用の成績や物価上昇率などのブレが異なる世界線をたくさん生成し、その中で何%成功するか」という数字が得られます。もちろん現実の世界というのは無数にある未来線のうちのたった一つなので、生存確率10%と言われても運が良ければ生きるかもしれませんし、90%でも破綻するかも知れません。ただ、確率を考える利点が２つあります。「2つの戦略のどちらが優れているかが議論できる」ことと、どんな戦略を立てても結局は「あなたが何%のプランなら満足するか」というところに帰着することにあります。

つまり、万人におすすめできるわかりやすい最適解というものはありません。その人のライフスタイルや満足度にかなり依存してしまいます。

### 最適戦略を求めてみた

このサイトでは、第3,4章で色々な戦略を試し、生存確率にどのような影響があるか精査しました。その上で、各年代ごとに取り崩し戦略として一番良いものを求め、分かりやすい公式を作りました。

## 第2章: 「4%ルール」を日本でそのまま当てはめてはいけない理由

アメリカで考案された元の4%ルールは、「S&P500と債券を75%:25%で持った場合の30年間生存確率」を元に組み立てられています。

**「資産を年率7%で運用し、インフレ率（物価上昇率）の3%を差し引いた、実質4%以内で取り崩せば、30年以上資産が枯渇しない」**という理由付けもよく見受けられます。

これを日本での FIRE戦略に当てはめる場合、いくつかの違いを考慮しなければなりません。

!!! warning "物価上昇率"
    シミュレーションにおいて、将来の物価上昇率（インフレ率）を考慮する必要があります。日本のインフレ率は違います。

!!! warning "譲渡所得税"
    日本では、投資の利益に対して約20%（20.315%）の譲渡所得税がかかります。資産を売却する際、この税金が支払われる分だけ手元に残る金額が減ります。

!!! warning "為替リスク"
    日本の投資家がオルカンなどの外貨建て資産を購入する場合、為替リスクが発生します。為替リスクは全体のリスク（ボラティリティ）を増加させます。

!!! warning "長生きリスク"
    あなたの年齢によりますが、30年生きる戦略と50年生きる場合で、違う戦略を取る必要があります。30年生きる場合には良いとされている戦略が50年の場合には逆になる可能性があります。

!!! warning "年金"
    年金を全く無視するシミュレーションはより安全ではありますが、最適戦略を考えるには年金も考慮に入れましょう。

## 第3章: 取り崩しシミュレーションにおいて考慮すべきこと

シミュレーションでは、固定利回りによる計算を排除し、現実の市場環境を反映させています。各要素を追加するごとに、破産確率は以下のように変化します。（初期資産1億円、毎年400万円を取り崩す設定）
<!-- COMMENT: なんかこの文章固い！-->

[ボラティリティが長期投資に与える影響](volatility.md)

!!! abstract "ボラティリティによる成長率の低下"
    * 算術平均リターンが7%であっても、ボラティリティ（15%）を考慮した中央値の成長率は**約5.8%**まで低下する。
    * 毎年7%で増え続けるという固定リターンの予測が実現する確率は、**約30%**しかない。
    * ボラティリティがない（リスク0%）なら破産確率は0%だが、ボラティリティ（15%）を加味するだけで、50年後の破産確率は**17%**になる。

[物価上昇率が取り崩しに与える影響](cpi.md)

!!! note "物価上昇率（1.77%）の追加"
    歴史的平均（2.44%）は1970年代の極端な物価高の影響を強く受けています。本プロジェクトでは、インフレの粘着性（自己相関）を考慮した詳細な分析に基づき、定常的な期待値である**「年率 1.77%」**を現実的な基準として採用しました。このインフレを考慮するだけで、50年破産確率は約18%から41%以上へと上昇します。

[譲渡所得税が取り崩しに与える影響](tax.md)

!!! note "譲渡所得税（20.315%）の追加"
    利益に対する税金の支払いを考慮すると、50年破産確率は 49.2% に上昇します。

[信託報酬が取り崩しに与える影響](trust_fee.md)

!!! note "信託報酬（0.05775%）の追加"
    インデックスファンドの運用コストを引くと、50年破産確率は 50.0% になります。

[為替リスクが取り崩しに与える影響](forex.md)

!!! note "為替リスク（10.5%）の追加"
    外貨建て資産の為替変動リスクをボラティリティとして加算すると、50年破産確率は 58.7% に達します。

[ファットテール分布が取り崩しに与える影響](fat_tails.md)

!!! note "ファットテール分布の影響"
    実際の市場では正規分布の予測を超える大暴落（ファットテール）が頻発します。しかし、長期的な期待リターンとボラティリティを正確に揃えた場合、50年間の超長期シミュレーションにおける破産確率への影響は限定的であり、対数正規分布モデルでも実用上は十分な精度の評価が可能です。

[購買力平価とインフレの相関](ppp.md)

!!! note "購買力平価（PPP）とインフレの関係"
    「円安になると輸入物価が上がり、インフレになる」という理論（購買力平価説）がありますが、過去45年間のデータを分析した結果、年次ベースでの為替変動とインフレ率の間に有意な相関は認められませんでした。シミュレーションでは両者を独立した要因として扱います。

[S&P500 vs オルカン 〜 取り崩し期における比較](sp500_vs_acwi.md)

!!! note "S&P500 vs オルカン"
    S&P500とオルカンの比較において最大のポイントは、「どの期間のデータでモデルを作るか」によって結果が大きく変わる点です。S&P500は直近30年間のリターンが155年の歴史的平均よりも好調でした。また、オルカンのデータが存在する直近18年間を基準にしても、米国株のパフォーマンスが他国を圧倒しており、オルカンよりもS&P500の方が破産確率が低く算出されます。将来何が正解になるかは誰にも分かりませんが、「実績のあるデータをどう信じるか」という仮説によって選ぶべき資産が変わります。

## 第4章: 生存確率を上げるための方法

破産確率を下げて生存確率を上げるために色々な戦略を試しました。

[現金を持つ効果](cash_ratio.md) ・ [現金とのリバランスの効果](cash_rebalance.md)
!!! failure "現金の保有は長期取り崩しには向かない"
    暴落への備えとして現金を保有し、それを優先的に消費することで、20年目付近までの生存確率をわずかに向上させることができます。しかし、30年以上の長期で見ると、現金比率が高いほど生存確率は低下します。現金がリターンを生まず、インフレによって実質価値が目減りするため、長期的な資産寿命を縮める要因となります。現金とのリバランス戦略は、成長しない現金へ定期的に資産を移すことになり、長期的な生存確率を下げます。

[無リスク資産を持つ効果](zero_risk.md) ・ [無リスク資産とのリバランスの効果](zero_risk_rebalance.md)
!!! success "利回りのある資産を優先的に消費して成長資産を守る"
    インフレ率を上回る実質リターンがある無リスク資産は、現金よりも資産を維持する効果に優れています。無リスク資産を先に取り崩すことで、運用初期の下落局面における成長資産の安値売りを抑制できます。50/50の配分では、取り崩しの順序を変えるだけで、50年後の破綻確率を77.9%から62.7%まで低減できることが確認されました。利回り（4%）のある無リスク資産とのリバランスは、30年程度の期間であれば生存確率を向上させる有効な手段となります。ただし、50年以上の超長期を目標とする場合は、次で紹介するダイナミックリバランスを行いましょう。

[ダイナミックリバランス戦略](dynamic_rebalance.md)
!!! success "毎年最適配分にリバランスしよう"
    株式と無リスク資産の配分を毎年最適配分する「ダイナミックリバランス」は生存確率を向上させます。初年度に最適配分した後比率を変えない戦略よりも平均で3.10%生存確率（パーセンテージポイント）が向上し、特に厳しい条件での長期運用で大きな差が出ます。例えば4%支出の場合の50年生存確率は 40.9%→55.9%へと向上します。

[ダイナミック・スペンディング（動的支出）](dynamic_spending.md)
!!! success "「相場が悪い時は支出を少し減らす」という工夫が、長期の生存に効く"
    運用状況に応じて、前年の支出額をベースに柔軟に予算を変える戦略です。好調時の増額を抑え（例：+1%）、不調時にわずかな支出削減（例：-1.5%）を受け入れるだけで、資産が尽きるリスクを大幅に低減できます。生存確率は「目標支出率」の設定に最も強く左右されますが、ダイナミックリバランスと組み合わせることで、生存確率はさらに10〜19%向上します。

[高齢になると支出が減る](retired_spending.md)
!!! warning "年齢による支出変動の影響"
    家計調査の統計データによれば、50代をピークとして、60代、70代と年齢が上がるにつれて消費支出が減少することが示されている。そのため、30代や40代でのリタイアでは、その後の数十年間にわたる支出の増加傾向が運用初期の資産を圧迫し、生存確率を大幅に低下させる。一方、50代後半以降のリタイアでは、加齢による支出の低下が取り崩し額を減らすため、結果として長期的な生存確率が高くなる。

[年金受給が取り崩し戦略に与える影響](pension.md)
!!! success "年金は生存確率を 15〜40% 以上向上させる"
    公的年金は、リタイア後の生活を支える基礎収入として重要です。特に年支出が少ない Lean FIRE では生存確率を 15%〜40% 以上向上させます。高支出でも5~15% の効果があります。65歳からの受給よりも、60歳からの繰り上げ受給の方が生存確率を伸ばしました。これは受給総額の損得よりも「運用初期の資産目減りを抑える効果」が大きいためです。また、早期リタイア時の免除制度活用や60歳での保険料支払い終了も、キャッシュフローを改善し生存確率を押し上げる重要な要因となります。

[サイドFIRE](side_fire.md)
!!! success "初期の労働収入が資産の維持を助ける"
    リタイア初期の5年間に生活費の半分を稼ぐだけで、50年後の資産を維持できる確率は約7〜10ポイント向上します。ダイナミックリバランスを併用すると、維持確率は70%以上に達します。短期間の安定を重視する場合は少額を長期間、長期的な資産の最大化を目指す場合は早期に集中して稼ぐのが合理的です。危険になったら働く方法も有効です。その場合一番重要なのは「働ける年数 × 稼げる量」で、次は「どれだけ年支出率が上がったら働くか」という閾値でした。

[資産が枯渇する前に死亡するケースを考えてみる](mortality.md)
!!! success "資産が枯渇する前に死んでしまうケースは意外に多い"
    50年間資産が破綻しない確率を真面目に求めても、日本人の年齢別死亡率を考慮するとそれより前に死んでしまうケースも実は結構あります。100歳まで生きられるのは男性の約1.5%、女性の約6.5%にすぎません。「資産が枯渇する前に死んだ場合は成功とみなす」と90歳時点での成功確率は男性で8〜10%、女性で5〜8%上がります。

## 第5章: 年齢別の最適戦略

==各ページには総資産に基づいた生存確率の公式を載せました。==

[60歳の最適戦略](all_60yr.md)
!!! success "6%ルールでも大丈夫なケースも"
    95歳まで生きる確率を最大化する戦略を紹介しています。60歳の人は支出も減っていき、年金も始まり、資金が枯渇しない目標年数も短いのでシミュレーション結果はとても良いものになっています。まずは年支出を決めるところから始めましょう。

[50歳の最適戦略](all_50yr.md)
!!! warning "4%ルールは少しだけ無理をしている"
    50歳から95歳までの45年間を生き抜く戦略です。4%ルールは大半の人にとって少しだけリスクが高く、最初の10年間の過ごし方が鍵となります。生存確率を最大化する公式も紹介しているので、差額分だけ生活を切り詰めたり副業で賄う方針も可能です。

[40歳の最適戦略](all_40yr.md)
!!! danger "55年間の超長期戦、4%ルールは危険"
    40歳から95歳までの55年間を生き抜くためには、4%ルールでは不十分なケースが多いことが分かりました。年金受給までの20年間、なるべく額面上の支出を増やさない我慢ができる場合の生存確率は高いです。ただし世帯人数が二人以上の一般家庭では40歳から55歳をピークに支出が伸びることと、物価上昇があることのダブルパンチがあるにもかかわらず、20年間も額面上の支出を増やさないことが現実的かは吟味しましょう。我慢できる場合の公式を載せました。

# 物価上昇率が取り崩しに与える影響

物価上昇は、お金の額面が変わらなくてもそのお金で買えるものの量が減る現象であり、長期の取り崩しにおいては将来の生活水準を左右する大きな要因となります。
<!-- COMMENT: 違う文がいいな-->

!!! abstract "重要なポイント"
    * **物価上昇は取り崩しにおいて極めて大きな影響を与える。** 年率2.44%の物価上昇を考慮するだけで、4%の取り崩し戦略の50年後の破産確率は約18%から51%以上へと大幅に跳ね上がる。
    * **インフレ率の毎年の変動（ボラティリティ）の影響はゼロではないが小さい。** 重要なのは平均的なインフレのトレンドであり、長期的なシミュレーションでは平均してどの程度の物価上昇が起こるかを見積もることが必須である。

## インフレ率（CPI）とは

物価上昇率（インフレ率）とは、ある期間において物価がどれくらい上がったかを示す割合のことです。シミュレーションにおいて将来の物価上昇率を予測する際、一般的には消費者物価指数（CPI）を用いて近似することが妥当とされています。

CPIは総務省統計局が毎月公表し、約700品目について基準年を100として指数化しています。私たちが日常生活で購入するモノやサービスの価格を幅広くカバーしているため、生活費の変動を測る上で適した指標です。ただし、医療費や介護費などはCPI全体の伸び率よりも高く上昇する傾向にあるため、高齢になるにつれて個人の生活費はCPI以上に上昇していると感じる可能性があります。

こうした前提を踏まえた上で、標準的な生活費の変動をモデル化するため、CPIの変動率を用いて物価上昇率を計算し、取り崩しに与える影響を検証します。

## 日本の物価上昇率の歴史

1970年から2025年までのCPIの実際のデータ（[政府統計の総合窓口 e-Stat](https://dashboard.e-stat.go.jp/)）を使用し、過去の物価変動の歴史を確認します。直近の動きを見ると、基準年である2020年を100とした場合、2025年には「112.0」まで上昇しています。つまり、たった5年間で私たちが普段買っているものの価格が平均して12%も上がったということであり、短期間でも物価変動が家計に与える影響は小さくありません。

![日本の物価上昇率の推移](imgs/cpi/historical_trend.svg)

過去55年分のデータから計算した日本の物価上昇率の統計値は以下の通りです。

*   **算術平均（年率）**: 2.44%
*   **標準偏差（ボラティリティ）**: 4.13%

高度経済成長期の高いインフレ率が含まれているため、平均値は2.44%とやや高めになっています。この歴史的な平均値とボラティリティを基準に、シミュレーションを行います。（初期資産1億円、初年度の取り崩し額400万円、オルカンに100%投資（リターン7%、ボラティリティ15%）の設定）

## 実験1：インフレ率（平均値）の影響

本シミュレーションの前提条件は以下の通りです。

!!! info "シミュレーションの設定"
    *   **初期資産**: 1億円を全てオルカン（期待リターン7%、ボラティリティ15%）に投資
    *   **取り崩し額**: 初年度は年間400万円（毎月約33.33万円のペースで取り崩し）
    *   **物価上昇の反映**: インフレ率に合わせて毎月の取り崩し額が徐々に増加していく設定（シミュレーション上は、設定した年率インフレ率を12分割した値を毎月の物価上昇率として適用し、複利で取り崩し額を増加させる）
    *   **その他**: 税金や信託報酬などの細かなコストは考慮しない

まず、インフレ率のボラティリティを0%とし、インフレ率の平均値（年率）のみを変化させた場合の影響を検証します。物価上昇に合わせて、毎年の取り崩し額を増加させた場合の、50年後の資産の分布と、10~50年後の破産確率はこちらです。

{!data/cpi/experiment1.md!}

詳細な生存確率のグラフはこちら。

![インフレ率の影響（生存確率）](imgs/cpi/experiment1_result.svg)

インフレ率が0%から歴史的平均の2.44%に上昇すると、50年後の破産確率は18.4%から50.9%へと大幅に跳ね上がります。物価上昇による生活費の増加は、長期の取り崩しにおいて極めて大きな影響を与えます。

## 実験2：インフレ率のボラティリティの影響

[ボラティリティが長期投資に与える影響](volatility.md)では、資産の価格変動（ボラティリティ）をシミュレーションに組み込むことが重要であることを確認しました。インフレ率にも年ごとの変動（ボラティリティ）が存在するため、これもシミュレーションに組み込むべきではないかという疑問が生じます。

そこで、インフレ率の平均値を2.0%に固定し、インフレ率のボラティリティ（毎年の物価上昇率のブレ幅）を変化させた場合の影響を検証します。

{!data/cpi/experiment2.md!}

詳細な生存確率のグラフはこちら。

![インフレ率のボラティリティの影響（生存確率）](imgs/cpi/experiment2_result.svg)

ボラティリティが0%の場合と、歴史的標準偏差である4.13%の場合を比較しても、50年後の破産確率は44.2%から43.4%へとわずかに変化するのみです。インフレ率の毎年の変動（予測の難しさ）自体は、長期的な取り崩しの成功確率にほとんど影響を与えません。

??? info "数学的な解説：なぜボラティリティが上がると破産確率がわずかに下がるのか"
    資産の成長と同じく、$\mu - \frac{\sigma^2}{2}$（ボラティリティ・ドラッグ）の数学的性質によるものです。

    インフレ率に変動（標準偏差 $\sigma$）がある場合、長期間経過後の物価水準の中央値は、算術平均のインフレ率 $\mu$ ではなく、それより少し低い幾何平均 $\mu - \frac{\sigma^2}{2}$ に収束していきます。

    例えば、インフレ率の平均値が2.0%（$\mu = 0.02$）、ボラティリティが4.13%（$\sigma = 0.0413$）の場合、幾何平均リターンは以下のようになります。

    $$0.02 - \frac{0.0413^2}{2} \approx 0.02 - 0.00085 = 0.01915 \text{ (約1.91%)}$$

    つまり、ボラティリティが4.13%あることで、長期的なインフレ率の中央値は2.0%から約1.91%へと「下押し」されます。物価の上昇ペースがわずかに鈍化するため、毎年の取り崩し額（生活費）の増加ペースも抑えられ、結果として資産の枯渇がわずかに遅れる現象が起きています。

    そして、2.0%から1.91%への変化はわずか0.1%未満の差であるため、結果に対する影響も「ほんの少し破産確率が下がるだけ」という微小なものに留まります。

## 結論

ライフプランのシミュレーションにおいて、将来の物価上昇率をどのように設定するかは、結果を大きく左右する極めて重要な決定事項です。毎年の物価の変動幅を正確に予測する必要はありませんが、平均してどの程度の物価上昇が起こるかを現実的に見積もることが必須です。
